# Get-GPORevWMI

## SYNOPSIS
Get Group Policy WMI filter

## SYNTAX

```
Get-GPORevWMI [[-Guid] <Guid[]>] [[-Name] <String[]>] [[-Forest] <String>] [[-ExcludeDomains] <String[]>]
 [[-IncludeDomains] <String[]>] [[-ExtendedForestInformation] <IDictionary>] [-AsHashtable]
 [<CommonParameters>]
```

## DESCRIPTION
Get Group Policy WMI filter

## EXAMPLES

### EXAMPLE 1
```
Get-GPORevWMI -AsHashtable
```
